Enviar para o carro

Este aplicativo permite que você envie um endereço do Google Maps diretamente para o sistema de navegação do seu veículo para que você não necessite digitar manualmente. Esta funcionalidade está disponível na versão web do Google Map, mas não na versão Android. Este aplicativo traz o recurso Send to Car carro para o Android. Para verificar se o modelo do seu veículo é suportado no seu país, visite o Google Maps no seu computador.

Como usar

Inicie o aplicativo Google Maps no seu dispositivo Android. Procure ou selecione um ponto de destino no mapa. Clique na bolha para visualizar detalhes sobre o ponto. Na parte inferior da página de detalhes, selecione "Compartilhar este lugar". Na lista de ações possíveis, selecionar "Sent To Car”.Uma vez que o endereço tenha sido carregado pelo Sent to Car, selecione seu veículo e o identificador da conta (este pode ser o nome de usuário que você usa para se conectar na página da internet do fabricante do seu veículo ou o número do seu telefone celular).

Uma vez que o endereço tenha sido carregado pelo Enviar para o carro, selecione seu veículo eo identificador da conta (este pode ser o nome de usuário que você usa para fazer login para web com o fabricante do veículo portal ou seu número de telemóvel).

Se você quiser, você pode editar o nome do destino antes de enviá-lo.

Em seguida, clique no botão "Enviar" está pronto para seguir em frente!
