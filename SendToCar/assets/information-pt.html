Enviar para o carro

Este aplicativo permite que você envie um endereço do Google Maps diretamente para o sistema de navegação do seu veículo para que você não tem que digitar manualmente-lo dentro Esta funcionalidade existe na versão web do Google Map, mas não na versão para Android . Este aplicativo traz o recurso Enviar para carro para o Android. Para ver se seu carro é suportado no seu país, visite o Google Maps no seu desktop.

Como usar

Inicie o aplicativo Google Maps no seu dispositivo Android. Pesquisar ou selecione um ponto de destino no mapa. Clique na bolha para visualizar detalhes sobre o ponto. Na parte inferior da página de detalhes, selecione "Compartilhar este lugar". Na lista de ações possíveis, selecionar "Enviar para o carro".

Uma vez que o endereço tenha sido carregado pelo Enviar para o carro, selecione seu veículo eo identificador da conta (este pode ser o nome de usuário que você usa para fazer login para web com o fabricante do veículo portal ou seu número de telemóvel).

Se você quiser, você pode editar o nome do destino antes de enviá-lo.

Em seguida, clique no botão "Enviar" e longe vai!
