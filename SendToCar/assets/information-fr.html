Envoyer à un véhicule

Cet application permet d'envoyer une destination directement au système de navigation de votre véhicule au lieu de devoir réentrer l'adresse manuellement. Cette fonction existe dans la version web de Google Maps, mais pas dans la version Android. Cet application ajoute la fonction Envoyer à un véhicule à la plateforme Android. Pour voir si votre véhicule est supporté dans votre pays, visitez Google Maps avec votre ordinateur de bureau.

Comment utiliser l'application

Démarrez Google Maps sur Android. Faites une recherche ou sélectionnez un endroit sur la carte. Cliquez sur la bulle pour avoir plus de détails sur l'endroit. En bas de la page de détails, sélectionnez "Partager". Dans la liste d'actions qui apparaît, sélectionnez "Envoyer à un véhicule".

Lorsque l'adresse est chargée, sélectionnez le constructeur de votre véhicule ainsi que le numéro de compte associé au véhicule (cela pourrait être le nom d'utilisateur que vous utilisez pour connecter au site web du constructeur, votre adresse de courriel ou votre numéro de téléphone).

Si vous le désirez, vous pouvez changer le nom de l'endroit avant de l'envoyer.

Finalement, cliquez sur "Envoyer" et la destination sera transférée à votre véhicule!
